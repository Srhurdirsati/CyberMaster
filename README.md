# CyberMaster
This is a custom ESP-32 based development board fully customized to enhance it's capabiities by little much . This will be having 17 GPIOs with a camera connector and with WiFi and Bluetooth , also accessible by USB C , Can be operated with battery too along with USB, I planned to create this a lil ago but due to ongoing exams , tomorrow is my mathemactics exam which I've already prepaerd for so I thought to make and finish this project in one shot today only . 

Started with schemtic and went all up to Routing within 12 Hours
